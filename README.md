# What does the project entail?
The project is a fun CLI tool that allows you to fetch recent activity of a GitHub User.
It is a [GitHub User Activity Homework](https://roadmap.sh/projects/github-user-activity) from [Roadmap.sh](https://roadmap.sh).

# How to use it?
Make sure to have .NET 9 installed.
Make sure to clone the repository:
```
git clone git@github.com:nikoloz-code23/GithubActivity.git
```
After that, inside the project directory, run:

```
dotnet run github-username
```
And that's about it.
